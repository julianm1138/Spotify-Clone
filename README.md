A Vite + TypeScript React application that replicates core Spotify functionality. Built primarily to demonstrate DevOps and cloud deployment skills, including containerization with Docker, CI/CD pipeline to Azure Container Registry (ACR), and live deployment on Azure Container Apps (ACA). The app showcases front-end capabilities while serving as a vehicle to demonstrate cloud infrastructure management, deployment automation, and monitoring.
